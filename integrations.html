TELEGRAM

Set up your Telegram bot.

5 steps. Takes about 5 minutes. No coding required.

01
Install EvoClaw (if you haven't)

Make sure EvoClaw is installed and the proxy is running on port 8080.

BASH
pip install evoclaw
evoclaw init
evoclaw start
✓ Verify proxy is running: curl http://localhost:8080/v1/models — should return a JSON response.
02
Create a bot via BotFather

BotFather is Telegram's official bot for creating other bots. Open Telegram and follow these steps:

1
Open Telegram → Search @BotFather
Find the verified BotFather bot (has a blue checkmark).
2
Send /newbot
BotFather will ask for a name and username for your bot.
3
Choose a name and username
Example name: EvoClaw Agent — username must end in _bot, e.g. evoclaw_agent_bot
4
Copy the token BotFather gives you
Looks like: 1234567890:ABCdef... — keep this safe, never share publicly.
03
Download the Telegram bot file

Get the bot script from the EvoClaw repo.

BASH
cd ~/evoclaw/repo
# File sudah ada jika kamu pull dari GitHub
ls evoclaw_telegram.py
✓ Kalau file belum ada: git pull origin main dari folder repo.
04
Install dependencies & jalankan

Install library yang dibutuhkan, lalu jalankan bot di screen supaya tetap hidup.

BASH
# Install dependencies
pip install python-telegram-bot openai --break-system-packages

# Jalankan di background (screen)
screen -dmS tg-bot bash -c "cd ~/evoclaw/repo && TELEGRAM_BOT_TOKEN='TOKEN_KAMU' python3 evoclaw_telegram.py"

# Cek sudah running
screen -ls
⚠ Ganti TOKEN_KAMU dengan token dari BotFather. Jangan share token di tempat publik!
05
Test bot kamu!

Buka Telegram, cari username bot yang kamu buat, dan kirim pesan pertama.

→
Kirim /start
Bot akan balas dengan pesan selamat datang.
→
Chat biasa
Kirim pesan apapun — bot akan balas dan agent belajar dari conversation ini.
→
Cek /stats
Lihat berapa conversations dan skills yang sudah agent pelajari.
✓ Agent sudah belajar ketika kamu lihat log di VPS: PRM score=0.xx | conv=N | skills=N
TELEGRAM COMMANDS

Available commands.

/start
Welcome message dan penjelasan cara pakai bot
/stats
Lihat statistik agent: skills, conversations, status Tinker LoRA
/skills
List semua skill categories yang sudah dipelajari agent
/reset
Hapus history conversation kamu — mulai fresh
/help
Tampilkan semua commands dan cara kerja EvoClaw
Pesan biasa
Chat langsung dengan agent — tidak perlu prefix apapun
DISCORD

Set up your Discord bot.

Lebih banyak fitur dari Telegram — embed cards, slash commands, dan per-channel memory.

01
Buat aplikasi di Discord Developer Portal

Pergi ke discord.com/developers/applications dan buat app baru.

1
Klik New Application
Kasih nama, contoh: EvoClaw
2
Klik Bot di menu kiri → Reset Token
Copy token yang muncul — simpan dengan aman.
3
Enable Privileged Intents
Scroll ke bawah di halaman Bot → aktifkan MESSAGE CONTENT INTENT dan SERVER MEMBERS INTENT → Save Changes.
02
Invite bot ke server Discord kamu

Generate invite URL dan add bot ke server.

1
Klik OAuth2 → URL Generator
Di menu kiri Developer Portal.
2
Centang Scopes: bot
Centang permissions: Send Messages, Read Message History, Use Slash Commands, View Channels.
3
Copy Generated URL → buka di browser
Pilih server kamu → klik Authorize.
INVITE URL (ganti CLIENT_ID)
https://discord.com/api/oauth2/authorize?client_id=CLIENT_ID&permissions=277025508352&scope=bot
03
Install dependencies & jalankan bot

Sama seperti Telegram — jalankan di screen supaya tetap hidup 24/7.

BASH
# Install dependencies
pip install discord.py openai --break-system-packages

# Jalankan di background
screen -dmS dc-bot bash -c "cd ~/evoclaw/repo && DISCORD_BOT_TOKEN='TOKEN_KAMU' python3 evoclaw_discord.py"

# Cek running
screen -ls
⚠ Ganti TOKEN_KAMU dengan token dari Discord Developer Portal. Jangan pernah share token!
04
Test di server Discord

Buka server Discord kamu dan test bot.

→
Mention bot: @EvoClaw halo!
Bot langsung balas dengan embed card.
→
Gunakan slash command: /ask apa itu LoRA?
Slash commands tersedia di semua channel.
→
Ketik !stats
Lihat statistik agent langsung di Discord.
DISCORD COMMANDS

Available commands.

@EvoClaw [pesan]
Mention bot di channel manapun — bot langsung balas
/ask [pertanyaan]
Slash command — tanya langsung ke agent
!stats
Lihat agent stats dalam embed card: skills, conversations, Tinker status
!skills
Tampilkan semua skill categories dalam embed card
!reset
Clear conversation history untuk channel ini
!help
Lihat semua commands dan cara kerja EvoClaw
/reset
Slash command untuk clear history (hanya terlihat oleh kamu)
TROUBLESHOOTING

Kalau ada masalah.

Solusi untuk masalah yang paling umum.

Bot tidak balas

Pastikan EvoClaw proxy running: evoclaw status

Cek bot screen masih hidup: screen -ls

Pastikan token benar dan tidak expired.

Error "proxy connection refused"

EvoClaw proxy tidak running. Jalankan: evoclaw start

Pastikan port 8080 tidak dipakai proses lain: ss -tlnp | grep 8080

Bot mati setelah disconnect SSH

Gunakan screen untuk run di background. Jangan jalankan langsung di terminal SSH.

Command: screen -dmS tg-bot bash -c "..."

Discord "Message Content Intent" error

Enable di Developer Portal: Bot → Privileged Gateway Intents → MESSAGE CONTENT INTENT → ON → Save Changes.

CEK SEMUA STATUS
# Lihat semua screen yang running
screen -ls

# Harusnya ada:
tg-bot         (Detached) ✅
dc-bot         (Detached) ✅

evoclaw-agent  (Detached) ✅

# Cek proxy
evoclaw status

# Masuk ke screen tertentu untuk lihat log
screen -r tg-bot
